# 📊 Sales Analytics Dashboard

## Project Overview
This project is an interactive **Sales Analytics Dashboard** built using **Microsoft Power BI**. I analyzed 10,000+ retail sales records to uncover key business insights.

## 🛠️ Tools Used
- **Power BI Desktop** - Data visualization and dashboard design
- **Excel** - Initial data exploration

## 📈 Key Business Insights
1. **Technology** products generated the highest revenue.
2. Sales peaked during **November and December** (Holiday season).
3. The **South region** had the lowest profit margins and needs marketing attention.
4. **Office Supplies** had high sales volume but surprisingly low profitability.
5. The **Corporate** customer segment brought in the most revenue.
